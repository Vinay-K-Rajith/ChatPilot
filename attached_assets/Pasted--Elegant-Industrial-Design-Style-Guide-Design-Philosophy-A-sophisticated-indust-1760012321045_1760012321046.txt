## **Elegant Industrial Design Style Guide**

### **Design Philosophy**
A sophisticated, **industrial-modern aesthetic** combining the sleekness of metallic surfaces with contemporary SaaS design patterns. Think **high-end manufacturing meets digital innovation** - clean, professional, and powerful.

---

### **1. Color Palette - Industrial Metals Theme**

**Primary - Steel Blues:**
- `#1E3A5F` - Deep steel blue (primary brand, headers)
- `#2C5F8D` - Medium steel blue (interactive elements)
- `#4A90BF` - Light steel blue (accents, hover states)
- `#E8F1F8` - Ice blue (subtle backgrounds)

**Secondary - Metallic Grays:**
- `#2D3748` - Charcoal gray (text, dark surfaces)
- `#4A5568` - Medium gray (secondary text)
- `#718096` - Silver gray (borders, dividers)
- `#A0AEC0` - Light gray (disabled states)
- `#E2E8F0` - Platinum (card backgrounds)
- `#F7FAFC` - Chrome white (page background)

**Accent - GMD Brand Colors:**
- `#D97706` - Industrial orange (warnings, alerts)
- `#059669` - Success green (completed, active)
- `#DC2626` - Alert red (errors, canceled)
- `#7C3AED` - Electric purple (premium features, CTAs)

**Gradient Overlays:**
- Steel gradient: `linear-gradient(135deg, #1E3A5F 0%, #2C5F8D 50%, #4A90BF 100%)`
- Metallic shine: `linear-gradient(180deg, rgba(255,255,255,0.1) 0%, rgba(255,255,255,0) 100%)`
- Dark industrial: `linear-gradient(135deg, #2D3748 0%, #1A202C 100%)`

---

### **2. Typography - Professional & Readable**

**Font Family:**
- Primary: `'Inter', 'SF Pro Display', -apple-system, sans-serif` (clean, modern)
- Monospace (for data): `'JetBrains Mono', 'Fira Code', monospace`

**Type Scale:**
- **Display**: 36px / Bold / -0.02em (hero headings)
- **H1**: 28px / Semibold / -0.01em (page titles)
- **H2**: 20px / Semibold / 0 (section headers)
- **H3**: 16px / Medium / 0 (card titles)
- **Body**: 14px / Regular / 0.01em (main content)
- **Small**: 13px / Regular / 0.01em (metadata)
- **Caption**: 11px / Medium / 0.02em (labels, uppercase)

**Text Colors:**
- Headings: `#2D3748` (charcoal)
- Body: `#4A5568` (medium gray)
- Secondary: `#718096` (silver gray)
- Muted: `#A0AEC0` (light gray)

---

### **3. Spacing System - Generous & Balanced**

**Scale (Based on 8px grid):**
- `4px` (0.5) - Minimal spacing
- `8px` (1) - Tight
- `12px` (1.5) - Compact
- `16px` (2) - Standard
- `24px` (3) - Comfortable
- `32px` (4) - Spacious
- `48px` (6) - Section breaks
- `64px` (8) - Large sections

**Component Spacing:**
- Card padding: 24px
- Section gaps: 32px
- Grid gaps: 20px
- Button padding: 12px 24px
- Input padding: 12px 16px

---

### **4. Gradient Headers - Signature Style**

**Hero/Cover Sections:**
```css
background: linear-gradient(135deg, #1E3A5F 0%, #2C5F8D 50%, #4A90BF 100%);
position: relative;
overflow: hidden;
```

**Glassmorphic Overlays:**
- Semi-transparent shapes floating on gradient
- `backdrop-filter: blur(40px)`
- Subtle white borders: `border: 1px solid rgba(255,255,255,0.2)`
- Soft shadows: `box-shadow: 0 8px 32px rgba(0,0,0,0.1)`

**Metallic Shine Effect:**
```css
&::after {
  content: '';
  background: linear-gradient(120deg, 
    transparent 0%, 
    rgba(255,255,255,0.3) 50%, 
    transparent 100%);
  animation: shimmer 3s infinite;
}
```

---

### **5. Card Design - Clean & Professional**

**Standard Card:**
- Background: `#FFFFFF`
- Border: `1px solid #E2E8F0`
- Border radius: `12px`
- Shadow: `0 1px 3px rgba(0,0,0,0.05)`
- Hover: `0 4px 12px rgba(0,0,0,0.08)` + slight lift

**Elevated Card (Stats):**
- Shadow: `0 4px 6px rgba(0,0,0,0.07)`
- Border: None or very subtle
- Hover: Scale slightly (1.02)

**Dark Card Variant:**
- Background: `#2D3748`
- Text: White/light gray
- Border: `1px solid rgba(255,255,255,0.1)`

---

### **6. Data Visualization - Industrial Precision**

**Chart Colors:**
- Primary data: Steel blue gradient
- Secondary: Metallic gray
- Positive: Success green
- Negative: Alert red
- Neutral: Silver gray

**Chart Style:**
- Clean grid lines: `#E2E8F0`
- Thick data lines: 3px
- Rounded line caps
- Subtle shadows on bars
- Metallic gradient fills

**Progress Bars:**
- Track: `#E2E8F0`
- Fill: Steel blue gradient with shine
- Height: 6-8px
- Border radius: Full
- Animated loading effect

---

### **7. Buttons - Industrial Strength**

**Primary Button:**
```css
background: linear-gradient(135deg, #2C5F8D, #1E3A5F);
color: white;
padding: 12px 24px;
border-radius: 8px;
font-weight: 600;
box-shadow: 0 2px 4px rgba(30,58,95,0.2);
transition: all 200ms;

&:hover {
  transform: translateY(-1px);
  box-shadow: 0 4px 8px rgba(30,58,95,0.3);
}
```

**Secondary Button:**
- Background: White
- Border: `2px solid #E2E8F0`
- Color: `#2D3748`
- Hover: Border color to steel blue

**Ghost Button:**
- Transparent background
- Hover: `rgba(44,95,141,0.08)` background

**Icon Button:**
- Square or circular
- Hover: Background + icon color change
- Size: 36-40px

---

### **8. Status Indicators - Clear Communication**

**Status Badges:**
```css
/* Active/Success */
background: rgba(5,150,105,0.1);
color: #059669;
border: 1px solid rgba(5,150,105,0.2);

/* In Progress */
background: rgba(44,95,141,0.1);
color: #2C5F8D;
border: 1px solid rgba(44,95,141,0.2);

/* Warning */
background: rgba(217,119,6,0.1);
color: #D97706;

/* Error/Canceled */
background: rgba(220,38,38,0.1);
color: #DC2626;
```

**Dot Indicators:**
- 8px diameter
- Positioned before text
- Can pulse with animation
- Colors match status theme

---

### **9. Data Tables - Industrial Grid**

**Table Design:**
- Header: Uppercase, 11px, `#718096`, letter-spacing 0.05em
- Row height: 60-72px (spacious)
- Border: Only horizontal borders (`#E2E8F0`)
- Zebra striping: Optional, very subtle (`#F7FAFC`)
- Hover: `#E8F1F8` (ice blue tint)

**Cell Styling:**
- Padding: 16px 20px
- Align numbers right
- Status badges inline
- Icons 18px, aligned with text

---

### **10. Forms & Inputs - Precision Tools**

**Input Fields:**
```css
background: white;
border: 2px solid #E2E8F0;
border-radius: 8px;
padding: 12px 16px;
font-size: 14px;
transition: all 200ms;

&:focus {
  border-color: #2C5F8D;
  box-shadow: 0 0 0 3px rgba(44,95,141,0.1);
  outline: none;
}
```

**Select Dropdowns:**
- Custom styled (hide default arrow)
- Steel blue arrow icon
- Dropdown menu with shadow
- Options have hover state

**Search Bar:**
- Icon inside (left-aligned)
- Placeholder: `#A0AEC0`
- Width expands on focus
- Clear button appears when typing

---

### **11. Navigation - Steel Framework**

**Sidebar:**
- Background: `#F7FAFC` or white
- Width: 260px
- Border right: `1px solid #E2E8F0`
- Items: 40px height, 8px radius
- Active: Steel blue gradient background with white text
- Hover: Light gray background

**Top Bar:**
- Background: White
- Border bottom: `1px solid #E2E8F0`
- Height: 64px
- Sticky positioned
- Subtle shadow on scroll

**Breadcrumbs:**
- Small text (13px)
- Separator: `/` or `â€º` in gray
- Current page: Bold, steel blue

---

### **12. Icons - Industrial Iconography**

**Style:**
- Outline/stroke icons (Lucide React)
- Stroke width: 2px
- Size: 20px (standard), 24px (large), 16px (small)
- Color: Inherit from text or steel blue

**Special Treatment:**
- Primary actions: Steel blue
- Destructive: Red
- Success: Green
- Neutral: Gray

---

### **13. Micro-interactions - Smooth & Professional**

**Animations:**
- Smooth transitions: 200-300ms
- Easing: `cubic-bezier(0.4, 0, 0.2, 1)`
- Subtle scale on hover (1.02-1.05)
- Fade in content: opacity + translateY

**Loading States:**
- Skeleton screens with shimmer (metallic shine)
- Spinners: Steel blue gradient
- Progress bars: Animated gradient fill

**Hover Effects:**
- Cards: Lift + shadow increase
- Buttons: Slight lift + shadow
- Table rows: Background color change
- Icons: Color shift + slight scale

---

### **14. Responsive Behavior - Adaptive Precision**

**Breakpoints:**
- Mobile: < 768px
- Tablet: 768-1024px
- Desktop: > 1024px
- Wide: > 1440px

**Mobile Adaptations:**
- Sidebar becomes drawer (overlay)
- Stats cards stack vertically
- Tables scroll horizontally or transform to cards
- Reduced padding (16px instead of 24px)

---

### **15. Accessibility - Professional Standards**

**Contrast Ratios:**
- Text on white: WCAG AAA compliant
- White on steel blue: AAA compliant
- All interactive elements: 4.5:1 minimum

**Focus States:**
- Visible focus ring (steel blue, 2-3px)
- Skip to content link
- Keyboard navigation support
- ARIA labels on icons

---

### **16. Special Effects - Industrial Polish**

**Shimmer/Shine:**
- Subtle metallic shine on gradients
- Animated on hover or loading states
- Diagonal sweep effect

**Depth & Layers:**
- Multiple shadow layers for depth
- Subtle inner shadows on inputs
- Elevation system (8 levels)

**Grid Patterns:**
- Subtle dot grid on backgrounds
- Blueprint-style lines (very faint)
- Technical drawing aesthetic
